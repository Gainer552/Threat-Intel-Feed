#Threat Intel Feed

#Overview

This Bash script fetches the latest headlines from specified security and hacking news feeds and displays them in a formatted output. The script is designed to provide a quick overview of current threat intelligence and security news.

#Features

- Fetches headlines from multiple RSS feeds.
- Displays the top 5 items from each feed.
- Supports customizable feed sources.
- Uses color-coded output for better readability.

#Prerequisites

- Bash shell environment.
- `curl` command-line tool.
- `grep`, `sed`, and `awk` utilities.

#Installation

1. Save the script to a file, for example: threat_intel_feed.sh.
2. Make the script executable: chmod +x threat_intel_feed.sh.
